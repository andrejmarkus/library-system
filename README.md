# Jednoduchý knižničný systém
Jednoduchý knižničný systém, implementovaný pomocou webového frameworku nazývaneho Flask. Cieľom tejto semestrálnej práce je ukázať jednú z možnosti tvorby webových aplikácií pomocou programovacieho jazyku Python. Zaoberá sa systémom, ktorý umožňuje prehľadávať katalóg knižnice a manažovať výpožičky. Slúži teda ako efiktívny a jednoduchý nástroj pre knižnicu.
## Funkcionality aplikácie
- Prehľadávanie katalógu knižnice
- Zobrazovanie dostupných kníh
- Zobrazenie výpožičiek
- Manažovanie katalógu kníh
- Priraďovanie výpožičiek
- Odoberanie výpožičiek
- Nastavovanie profilu
- Prístup na báze rolí (použivateľ/admin)
## Použité zdroje
- **Python:** Programovací jazyk použitý na backend webovej aplikácie
- **HTML/CSS/JavaScript**: Jazyky použité na forontend webovej aplikácie
- **Flask:** Framework pre tvorbu webovej aplikácie
- **Bcrypt:** Knižnica na šifrovanie hesiel
- **MongoDB:** Databáza používaná na ukladanie dát
- **MongoEngine:** Knižnica, ktorá funguje ako ODM pri komunikácií s MongoDB databázou
